结果分析

如何使用结果分析？

1. 选择任务：在左侧任务列表中点击已完成的回测任务

2. 查看概览：右侧会显示该任务的关键指标和收益曲线

3. 对比分析：可以选择多个任务进行对比分析

4. 下载报告：点击任务卡片中的下载按钮获取详细报告
